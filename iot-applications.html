what is an iot application ?

iot application is an collection of service where we can intwgrate data from various devices and sensors. it can work in real-time, and
informed decisions can also be made by it.

what are main features needed?

the main features of Iot App is they should work in real-time, person should be able to control it in real-time from any where. secondly, app
should be stable and secure, so that it could work properly and secure from users. moreover, intregration of devices in Iot should also be
smooth such that, user can easily integrate device in app.

what are some examples?

  • self driven cars

  • Home Automation
  • Automated Transportation
  • Agricultural Drones

smart refers to the device which is able to perform things intelligently, it is also logically efficient and folloews percipient approach.

what are the main features needed?

features needed in smart building are automated lightning system, smart taps, Automated doors, Accessibility for the disables
and Automated parking system

what are some Smart-building scenarios?

  • Carbon-monoxide Monitoring to maintain appropriate AQI.
  • HVAC- Controlling Building Temperature
  • Security and Maintenance through devices to keep a check on exits and CCTV remotely.
  • Detect Occupancy - the presence of movement within its given range.
